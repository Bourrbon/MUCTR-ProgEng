@startuml
actor Клиент
actor Оператор
participant "Система" as Система

Клиент -> Оператор : Поиск еды
Оператор -> Система : Запрос списка блюд
Система --> Оператор : Список блюд
Оператор --> Клиент : Отображение списка блюд

Клиент -> Оператор : Выбор блюда
Оператор -> Система : Запрос информации о блюде
Система --> Оператор : Информация о блюде
Оператор --> Клиент : Информация о блюде

Клиент -> Оператор : Запрос на оформление заказа
Оператор -> Система : Запрос подтверждения заказа
Система --> Оператор : Подтверждение заказа
Оператор --> Клиент : Подтверждение заказа

Клиент -> Оператор : Оплата заказа
Оператор -> Система : Отправка чека и информации о заказе
Система --> Оператор : Подтверждение получения

Оператор --> Клиент : Уведомление о доставке
@enduml
